Solución: sincronizar antes de empujar
1. Primero haz un pull para traer los cambios remotos

git pull origin main --rebase

2. Resuelve cualquier conflicto si aparece
Si Git te muestra conflictos, abre los archivos afectados, resuélvelos manualmente, luego:

git add .
git rebase --continue

3. Ahora sí, haz el push

git push origin main
